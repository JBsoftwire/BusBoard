Machine Learning Algorithms

  • Linear models
  • Decision trees
  • Neural networks
  • Support vector machines

Linear Models

Unglamorous but very well-understood, easy to interpret, versatile (if you use a GLM and/or nonlinear features) and tbh probably best for whatever it is you're doing.

Decision Trees

These are very easy to interpret but not very good, unless you use an ensemble in which case they aren't easy to interpret any more. Glamorous ensemble types are random forests and boosting.

Neural Networks

Could uncharitably be described as a bunch of (generalised) linear models taped together. The most glamorous. Let you use cool terms like 'deep learning' and 'convolutional'.

Support Vector Machines

Honestly I don't really get how these work, luckily I never had an interviewer ask.
